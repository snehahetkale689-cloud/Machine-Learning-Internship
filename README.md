# Cognifyz Technologies - Machine Learning Internship

## About

This repository contains the projects completed during my Machine Learning Internship at Cognifyz Technologies. The internship provided hands-on experience in data preprocessing, exploratory data analysis, regression, and classification using Python and Scikit-learn.

## Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Task 1: Data Exploration and Preprocessing

### Objectives
- Loaded and explored the dataset
- Checked dataset dimensions and data types
- Identified and handled missing values
- Performed data cleaning
- Generated descriptive statistics
- Explored categorical and numerical features

### Skills Learned
- Data Cleaning
- Data Preprocessing
- Exploratory Data Analysis (EDA)

## Task 2: Regression Analysis

### Objectives
- Selected input and target variables
- Split the dataset into training and testing sets
- Built regression models
- Evaluated the models using Mean Squared Error (MSE) and R² Score
- Compared model performance

### Skills Learned
- Regression
- Model Evaluation
- Performance Analysis


## Task 3: Cuisine Classification

### Objectives
- Preprocessed the dataset
- Encoded categorical variables
- Built a Random Forest Classifier
- Predicted restaurant cuisines
- Evaluated the model using Accuracy, Classification Report, and Confusion Matrix
- Analyzed Feature Importance

### Results
- Model: Random Forest Classifier
- Accuracy: Approximately 31.55%

### Skills Learned
- Classification
- Random Forest
- Feature Importance
- Model Evaluation
